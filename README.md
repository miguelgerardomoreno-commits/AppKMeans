# Aplicación Flask de clasificación de clientes

Esta aplicación recibe ingresos y gastos de un cliente y devuelve el grupo asignado usando un modelo KMeans entrenado con datos de ejemplo.

Instalación (recomendado en virtualenv):

```bash
python -m venv .venv
.venv\Scripts\activate    # Windows PowerShell
pip install -r requirements.txt
```

Ejecutar:

```bash
python app.py
```
